# SARL Immobilières Algérienne - Site Vitrine

Site vitrine moderne et professionnel pour SARL Immobilières Algérienne, spécialisée dans la promotion immobilière à Oran.

## 🏢 À Propos

SARL Immobilières Algérienne est une entreprise leader en promotion immobilière à Oran, offrant des solutions d'habitation modernes et des services immobiliers complets.

## 🌟 Caractéristiques

✨ **Design Moderne** - Interface élégante et professionnelle
📱 **Responsive** - Adapté à tous les appareils (mobile, tablette, desktop)
⚡ **Performance** - Chargement ultra-rapide
🎨 **Palette Professionnelle** - Couleurs inspirées du secteur immobilier
🔄 **Navigation Fluide** - Scroll lisse et transitions animées

## 🚀 Déploiement

### Sur Vercel (Recommandé)

1. **Via GitHub Integration**
   - Pushez le code vers GitHub
   - Connectez à Vercel via GitHub
   - Déploiement automatique !

2. **Via Vercel CLI**
   ```bash
   npm install -g vercel
   vercel
   ```

3. **Via Drag & Drop**
   - Allez sur https://vercel.com
   - Drag & drop votre dossier
   - Déploiement instant !

[📖 Guide de Déploiement Complet](./DEPLOYMENT_GUIDE_FR.md)

## ⚙️ Configuration

Le fichier `vercel.json` configure Vercel pour servir le site statique:

```json
{
  "buildCommand": "",
  "outputDirectory": ".",
  "framework": "html"
}
```

## 📝 Modifications Possibles

Pour personnaliser le site:

1. **Changer les couleurs** - Modifiez les variables CSS en haut du style
2. **Mettre à jour le contenu** - Éditez le texte dans les sections
3. **Ajouter des images** - Insérez vos images et mettez à jour les liens
4. **Changer le logo** - Remplacez `logo.png`
